Place Smiley Sans (得意黑) variable font here.

Required file:
- SmileySans-Oblique.woff2 (variable font)

Download from the official repository releases:
https://github.com/atelier-anchor/smiley-sans/releases

After placing the file, it will be served at:
/fonts/smiley-sans/SmileySans-Oblique.woff2
